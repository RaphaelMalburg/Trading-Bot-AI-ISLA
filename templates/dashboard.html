How ISLA Bot Works
1
Market Data
Alpaca API pulls {{ timeframe_hours }}H OHLCV candles for BTC/USD (90-day window)
›
2
Feature Engineering
17 technical indicators computed — RSI, MACD, EMA, Bollinger Bands, ATR, OBV and more
›
3
ML Prediction
Random Forest classifies: will TP be hit before SL within {{ bot_config.max_hold_candles }}H? → LONG or FLAT + confidence
›
4
Sentiment Gate
News headlines scored –1 → +1 by Gemini LLM. Blocks entry if market is very bearish
›
5
Risk Controls
Circuit breaker · ATR-based position sizing · {{ (bot_config.max_risk_per_trade * 100)|int }}% max risk per trade · {{ (bot_config.daily_loss_limit_pct * 100)|int }}% daily loss limit
›
6
Execution
Market BUY · stop-limit SL at {{ bot_config.sl_atr_mult }}×ATR below entry · limit TP at {{ bot_config.tp_atr_mult }}×ATR above entry
Trade Trigger Logic
IF prediction == LONG AND confidence ≥ {{ (bot_config.confidence_threshold * 100)|int }}% AND sentiment ≥ {{ bot_config.sentiment_floor }} AND no open position
→ BUY with {{ (bot_config.max_risk_per_trade * 100)|int }}% risk per trade · SL = entry − {{ bot_config.sl_atr_mult }}×ATR · TP = entry + {{ bot_config.tp_atr_mult }}×ATR
→ CLOSE if already positioned and signal flips to FLAT, or SL/TP/max-hold hit (app-side watchdog every 10s)
Full Decision Flow — What Happens at Every {{ timeframe_hours }}H Candle Close
1
Market Data Fetch
Alpaca API returns the last 90 days of {{ timeframe_hours }}H OHLCV candles for BTC/USD (~2,160 bars). The most recent closed bar is used as the decision bar — the current forming candle is ignored.
2
Feature Engineering (17 indicators)
Computed on every bar: RSI-14 MACD MACD signal MACD hist EMA-20 EMA-50 BB high/low/width ATR-14 OBV Volume SMA ratio Log return Candle body ratio EMA-20 slope Dist EMA-20 Dist EMA-50. All features are StandardScaler-normalized before prediction.
3
ML Prediction
Random Forest (200 trees, max depth 8) outputs a probability for label 1 = "TP will be hit before SL within {{ bot_config.max_hold_candles }} candles ({{ bot_config.max_hold_candles * bot_config.timeframe_hours }}H)." If probability ≥ {{ (bot_config.confidence_threshold * 100)|int }}% → signal = LONG. Otherwise → FLAT. Top features: OBV, ATR-14, EMA-20, EMA-50 (see chart below).
4
Drift Detection
Checks the average confidence of the last {{ bot_config.max_hold_candles }} runs. If it falls below 50%, a drift warning is raised on the dashboard — the model may need retraining.
5
Sentiment Gate
{% if bot_config.sentiment_floor is defined %} Recent BTC headlines are fetched via Alpaca news and scored –1 → +1 by Gemini Flash LLM. A score below {{ bot_config.sentiment_floor }} blocks entry regardless of ML signal (avoids buying into strongly bearish news events). If the OpenRouter API key is not set, this gate is skipped. {% endif %}
6
Circuit Breaker
Checks today's realized P&L from the database. If cumulative daily loss exceeds {{ (bot_config.daily_loss_limit_pct * 100)|int }}% of starting equity, all trading halts for the rest of the day.
7
Position Sizing
Fixed-fraction risk: position_value = equity × {{ (bot_config.max_risk_per_trade * 100)|int }}% ÷ stop_pct where stop_pct = ATR × {{ bot_config.sl_atr_mult }} ÷ price. Result is clamped between 5% and 98% of available buying power.
8
Execution & Exit Management
Entry: market BUY on Alpaca + a disaster stop limit order at {{ bot_config.sl_atr_mult * 3 }}×ATR below entry (crash safety net only).
Exit (app-side, every 10s): SL close at {{ bot_config.sl_atr_mult }}×ATR below entry · TP close at {{ bot_config.tp_atr_mult }}×ATR above entry · Max hold close after {{ bot_config.max_hold_candles }} candles ({{ bot_config.max_hold_candles * bot_config.timeframe_hours }}H) · Signal flip close if FLAT at next candle close.
